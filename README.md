# aspiora liquid glass ui

a chill take on apple's liquid glass design language. built entirely with vanilla js, css, and tailwind v4. 

it simulates physical light reflections, dynamic lensing, and semantic vibrancy. basically, the cards float, tilt in 3d space, and shine when you hover over them.

## features

- lensing: real-time background blurring and high saturation to give it a deep, physical presence.
- highlights: dynamic specular reflections that actually respond to your mouse position to mimic real glass catching light.
- vibrancy: smart color mixing so the background gradients bleed through the cards and typography smoothly.

## running it locally

just clone the repo, install the dependencies, and start the dev server.

```bash
npm install
npm run dev
```

everything is neatly organized so you can just dive in and tweak the css/js to your liking. feel free to drop it into your own stuff.
